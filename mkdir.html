명령어 - mkdir

mkdir

디렉터리(폴더) 생성

파일 or 디렉터리가 동일명으로 존재할 경우 mkdir 명령어로 디렉터리 생성 불가

생성할 디렉터리의 경로 중 없는 경로로 디렉터리 생성 시, 생성 불가

경로에 f가 없기 때문에 g라는 디렉터리 생성 불가

옵션

-p

-p(parents)

필요한 경우 -p 옵션을 이용해 부모 디렉터리도 같이 생성
