디렉터리 구조

ls /

list의 줄임말로 현재 위치나 특정 경로의 디렉토리 내용의 리스트를 출력

구분 Linux Window
경로 / \
명칭 디렉터리 폴더
/ C:\
/home/ : 사용자 계정의 홈 디렉터리들의 기본 모임 C:\사용자\ or C:\Users\
최상위 경로 /root/ : 관리자 홈 디렉터리
/etc/ : 시스템 및 설치된 프로그램들이 저장 Windows랑 비슷한 역할
/bin -> /usr/bin/ : 기본 명령어 실행 프로그램들
/sbin -> /usr/sbin : 시스템 명령어들이 있다.(root)만 사용
관리자 권한이 있어야 사용
