명령어 - cp

cp

파일이나 디렉터리를 복사, 붙여넣기(기본 : 파일만)

문법

### 하나씩 복사 붙여넣기

cd -[옵션] <원본> <목적지>


### 여러 개 복사 붙여넣기(개수 제한 X)

### 이름 변경 불가능

cp -[옵션] <원본1> <원본2> ... <목적지>

목적지 표현 방법 두 가지

1. 원본 이름 생략해서 복사 붙여넣기

cp [복사할 파일] [붙여넣을 위치]


2. 원본 이름을 변경해서 복사 붙여넣기

cp [복사할 파일] [붙여넣을 위치]/[변경할 파일 이름]

이름 변경해서 붙여넣기 할 시, 중복된 명일 경우(파일명이 존재할 경우)

같은 경로에 동일한 이름이 있을 경우 강제 덮어 씌우기가 된다.

옵션

-r

복사&붙여넣기 시, 디렉터리도 포함하기 위해서 사용

/backup/ 디렉터리를 ../test/c 경로 아래로 붙여넣기

../test/c 아래에 backup 디렉터리랑 안에 내용도 다 붙여넣기 된 것을 확인
