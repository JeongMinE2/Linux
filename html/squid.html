Squid란?

오픈 소스(GPL) 소프트웨어 프록시 서버이자 웹 캐시

기본 포트는 3128

반복된 요청을 캐싱함으로 웹 서버의 속도를 향상 시킨다.

네트워크 자원을 공유하려는 사람들에게 웹, DNS와 다른 네트워크 검색의 캐싱을 제공한다.

트래픽을 걸러줌으로써 안정성에 도움을 주는 등에 이르기까지 광범위하게 이용된다.

예시

패키지 설치

# apt install -y squid

설정 파일 수정

# vi /etc/squid/squid.conf

acl [차단할 별칭] [IP]/[서브넷 마스크]
http_access allow [차단할 별칭]
cache_dir ufs /var/spool/squid 1000 16 256

ACL(Access Control List)

특정 이용자에 대한 접근 허용 또는 차단할 때 사용

시스템 재시작

# systemctl restart squid

윈도우 켜고

설정에서 '프록시' 검색

리눅스로 돌아가서

시스템 중단

# systemctl stop squid
