명령어 - find

# find

주어진 경로 조건부터 검색하여 이름을 찾는다.

문법

find [경로] -name "찾을 이름"

경로는 절대 경로 사용(상대 경로 사용 금지)

“ “, ‘ ‘ 둘 다 사용 가능

### -type d : 디렉터리만 찾기

find [경로] "찾을 이름" -type d


### -type f : 파일만 찾기

find [경로] "찾을 이름" -type f

최상위 경로로부터 'samadal'이라는 이름을 검색

최상위 경로로부터 'samadal'이라는 이름이 들어가는 것을 검색하고 줄 번호를 붙여서 출력
