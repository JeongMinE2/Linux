vi 편집기

# vi

빈 문서 실행(즉, 문서 파일 작업)

Emacs와 함께 유닉스 환경에서 많이 쓰이는 문서 편집기

명령, 실행(EX), 입력 모드가 있다.

문법

vi [편집할 파일명]

지정한 파일을 열고 편집

:set nu

vi 편집기에서 줄 번호를 활성화 시켜준다.

:[숫자]

vi 편집기에서 원하는 줄 번호로 이동

없는 파일을 vi 편집기로 실행시키고

:wq로 저장하고 끄면 새로운 파일이 생긴다.

vi의 기본 모드로, vi가 처음 실행되거나 입력 모드에서 ESC 키를 누른 경우

커서 이동, 문자열 수정, Copy & Paste 등

커서 이동

Key 동작
h 커서를 왼쪽으로 이동
j 커서를 아래로 이동
k 커서를 위로 이동
l 커서를 오른쪽으로 이동

커서 이동(행 단위)

Key 동작
0(Zero) 행의 처음으로 이동
$ 행의 마지막으로 이동

커서 이동(문서 단위)

Key 동작
gg 문서의 처음으로 이동
G 문서의 마지막으로 이동
Key 동작
x 커서가 있는 문자 삭제(delete 키와 같음)
X 커서가 있는 앞 문자 삭제(Backspace 키)
dd 현재 커서의 행 삭제
숫자 + dd 현재 커서부터 숫자만큼 행 삭제
yy 현재 커서의 행 복사
숫자 + yy 현재 커서부터 숫자만큼의 행 복사
p 복사한 내용을 현재 행 이후에 붙여넣기
P 복사한 내용을 현재 행 이전에 붙여넣기
u 되돌리기(undo) / 앞으로(redo)

명령 상태에서 키를 누르면 진입

왼쪽 하단에 --INSERT-- 라고 뜬다.

Key 동작
i 커서의 현재 위치에서 입력 모드로 변경
a 커서의 현재 위치에서 우측으로 한 칸 이동 후 입력 모드로 변경
o 커서 아래에 새로운 행을 추가하고 입력 모드로 변경
O 커서 위에 새로운 행을 추가하고 입력 모드로 변경

검색

/

패턴이 검색된 후 n 키를 통해 아래 방향으로 계속 찾기

패턴이 검색된 후 N 키를 통해 위 방향으로 계속 찾기

치환

:[범위]s/[Old]/[New]/[옵션]

Old라는 단어를 New로 치환


띄어쓰기 없이 다 붙여서 작성

범위는 [n(숫자)] 또는 [n, n] 또는 [%]를 넣을 수 있다.(범위는 n(숫자)이랑 %만 기억해도 충분)

s : 치환 한다.

/ : 구분자

옵션

g

g 옵션을 주면 적용되는 라인(커서가 있는 위치의 줄)의 모든 부분 변경

g 옵션을 주지 않으면 처음 찾은 부분만 변경

vi 편집기 일시중지

:! [명령어]

vi 편집기 일시중지하고 명령어 수행(작업 최소화)

명령문 이어서 작성

:! [명령문]; [명령문]

; 뒤에 명령문을 작성하면

; 전에 명령문을 실행한 뒤, 이어서 ; 다음 명령문이 실행된다.

Key 동작
:e 파일 열기(현재 작업 중인 vi 편집기 창이 닫혀야 가능)
:e! [파일명] 현재 작업 중인 vi 편집기를 저장하지 않고 강제 열기
:enew 현재 작업 중인 vi 편집기를 닫고 빈 문서 열기
:q 종료(변경된 내용이 없는 경우)
:q! 강제 종료(변경된 내용이 있어도 무시)
:w 현재 작업 중인 vi 편집기 파일 저장
:w [파일명] 현재 작업 중인 vi 편집기 다른 이름으로 저장
:wq 현재 작업 중인 vi 편집기 파일 저장 후 종료
Key 동작
:f 현재 작업 중인 파일의 이름과 라인 수 출력
:[n]r [파일명] 파일의 내용을 현재 편집 중인 파일의 n번 째 라인부터 삽입
:[n]r! [명령문] 명령문 실행 결과를 파읠의 n번 째 라인부터 삽입
