명령어 - ls

ls

디렉터리 안에 있는 내용물 나열(파일의 경우엔 파일 자체가 확인)

문법

ls [옵션] <대상1> <대상2> ...

대상을 생략할 경우 현재 위치가 디폴트

옵션

-a

.으로 시작하는 ‘숨겨진 것들도 포함’해서 모두 나열

-l

경로 안에 있는 것들의 속성 정보 자세히 확인

-R

경로 하위에 있는 세부 디렉터리들도 모두 자동 나열

예시

최상위 아래에 home이라는 경로랑 현재 위치를 확인
